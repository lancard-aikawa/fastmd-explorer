## 1. フォルダを開く

### パスを直接入力する
ヘッダーのパス入力欄にフォルダのパスを入力して `Enter` または「開く」ボタンを押します。

### フォルダ参照ダイアログを使う
`📂` ボタンをクリックするとOS標準のフォルダ選択ダイアログが開きます。

### 履歴から選ぶ
「履歴 ▾」ドロップダウンに過去に開いたフォルダが記録されています。選択するとそのまま開きます。

### ドラッグ＆ドロップ
フォルダをブラウザウィンドウにドロップしても開けます。

### 起動時に指定する（起動引数）
起動時にフォルダを引数で渡すと、そのフォルダを開いた状態で起動します。

```
fastmd-explorer.exe C:\docs
pnpm start /path/to/your/docs
```

指定したパスが存在しない場合や、フォルダでない場合は警告を出し、従来どおり前回の状態で起動します。
`https://` で始まる引数を渡した場合は [URLモード](#2-urlモードリモート-md-を開く) で起動します。

> Markdown ファイルが 1 件もないフォルダはツリーに表示されません。
> 履歴・フォルダ参照・起動引数のいずれで開いた場合も毎回スキャンし直すため、アプリを閉じている間のファイル増減も反映されます。

## 2. URLモード（リモート md を開く）

Web 上に置かれた Markdown ファイルを、ダウンロードせずにそのまま閲覧できます。

### モードを切り替える
ヘッダー左端の「フォルダ / URL」トグルで切り替えます。フォルダモードのままパス入力欄に `https://...` を入力しても、自動的に URL モードに切り替わって開きます。

### URL 履歴
開いた URL は履歴に保存され、サイドバーの一覧（クリックで再表示）と「履歴 ▾」ドロップダウンの両方から開き直せます。

### ワークスペースの独立
フォルダモードと URL モードは別々のワークスペースとして扱われます。モードを切り替えても、それぞれで開いていたタブがそのまま保持されます。

### リンクの扱い
プレビュー内の `.md` リンクはそのウィンドウ内でたどれます（タブごとの戻る・進むも有効）。それ以外のリンクは別タブで開きます。

> URL モードは**読み取り専用**です。編集・タグ・フラグ・メモ・被参照の表示は使えません。
> 取得サイズの上限は 5MB、タイムアウトは 15 秒です。

## 3. ファイルツリー

左のサイドバーにフォルダ構造とファイル一覧が表示されます。

### ディレクトリの展開・折り畳み
フォルダ行をクリックすると展開・折り畳みを切り替えられます。

### キーボード操作
ツリーをクリックするとフォーカスが移り、矢印キーで操作できます。

| キー | 動作 |
|---|---|
| `↑` / `↓` | カーソルを上下に移動（折り畳まれた中の行は飛ばします） |
| `Enter` | カーソル位置を開く（フォルダは展開・折り畳み） |
| `→` | 折り畳まれたフォルダを展開 |
| `←` | 展開されたフォルダを折り畳み |

### ルートフォルダ行
ツリー最上部のフォルダ名（太字・アクセントカラー）はルートフォルダを示します。右クリックでルート直下に新規作成できます。

### フォルダバッジ
フォルダアイコンの横に `md` / `img` / `html` のバッジが表示されます。それぞれ Markdown ファイル・画像ファイル・HTML ファイルを含んでいることを示します（サブフォルダを含む再帰的な判定。`html` バッジは「HTML」表示がオンのときだけ表示されます）。

### 画像・HTML の表示切替
サイドバー上部の「画像」「HTML」ボタンで、ツリーに画像ファイル・HTML ファイルを表示するかを切り替えます（既定はどちらもオフ、選択は記憶されます）。詳細は [13. 画像の表示](#13-画像の表示) / [14. HTML ファイルの表示](#14-html-ファイルの表示) を参照してください。

### ファイルカウント
サイドバー下部に総ファイル数が表示されます。フィルタ中は「表示件数 / 総数」になります。

### ツリーの再読み込み
ヘッダーの `↺` ボタン、サイドバー上部の `↺` ボタン、または `Ctrl+R` でツリーを再スキャンします。外部でファイルを追加・削除した場合に使います。

## 4. ファイルを開く・タブ操作

### ファイルを開く
ファイル行をクリックするとタブで開きます。

### タブの切り替え
タブバーのタブをクリックして切り替えます。`Ctrl+Tab` / `Ctrl+Shift+Tab` でも切り替えられます。

### タブを閉じる
タブの `×` ボタン、または `Ctrl+W` で閉じます。

### 全タブを閉じる
タブバー右端の「全て閉じる」ボタンで一括クローズします。未保存の変更があれば確認ダイアログが表示されます。

### ナビゲーション履歴（戻る・進む）
プレビュー内のリンクをたどると、タブごとに閲覧履歴が記録されます。ファイルヘッダー左側の `←` / `→` ボタン、または `Alt+←` / `Alt+→` で前後のファイルに移動できます。履歴はタブごとに独立し、タブを閉じると破棄されます。

### タブ状態の保存
フォルダを切り替えても、戻った時に以前のタブが復元されます（`localStorage` に保存）。

## 5. プレビュー表示

ファイルを開くと Markdown がレンダリングされて表示されます。

### Mermaid ダイアグラム
コードブロックに` ```mermaid ` で記述したダイアグラムが自動でレンダリングされます。

- **ズーム**: ダイアグラム上でマウスホイールを回す
- **パン（移動）**: ダイアグラムをドラッグ
- **ズームリセット**: ダイアグラム右上の `↺` ボタン
- **拡大・縮小ボタン**: ダイアグラム右上の `+` / `−`

### コードハイライト
コードブロックはシンタックスハイライト表示されます（テーマに連動）。

### コードブロックのコピー
コードブロックにマウスを合わせると右上に「コピー」ボタンが表示されます。クリックでクリップボードにコピーされ、「✓」表示に変わります。

### テーブルのソート
テーブルのヘッダー行をクリックするとその列で昇順・降順ソートできます。`↑` / `↓` が列名の後ろに表示されます。

### Wiki リンク
`[[ファイル名]]` または `[[ファイル名|表示名]]` 形式で他の Markdown ファイルへリンクできます。プレビュー内ではクリック可能なリンクとしてレンダリングされ、クリックでそのファイルを開きます（`.md` は自動補完されます）。

### ファイル情報バー
プレビュー上部に「更新: 〜　文字数: 〜　見出し: 〜」のファイル情報バーが表示されます。最終更新日時・本文の文字数・見出し数を確認できます。

## 6. アウトライン（TOC）パネル

ファイルに見出し（`h1`〜`h4`）があると、プレビュー右側にアウトライン（目次）パネルが表示されます。見出しが 1 件も無いファイルでは自動的に非表示になります。

### 見出しへジャンプ
アウトラインの項目をクリックすると、対応する見出しまでスムーズスクロールします。

### パネル全体の開閉
パネル左端の `◀` / `▶` ストリップをクリックすると、アウトラインパネル全体を折り畳めます。開閉状態は記憶されます（`localStorage`）。

### 見出し単位の折り畳み
子見出しを持つ項目には `▼` / `▶` トグルが表示されます。クリックすると配下の見出しを折り畳み・展開できます（この折り畳み状態はファイルを開き直すとリセットされます）。

### パネル幅のリサイズ
プレビューとアウトラインの境界をドラッグして幅を調整できます。境界をダブルクリックすると自動幅に戻ります。

> 編集の分割表示中はアウトライン（TOC）パネルは非表示になります。

## 7. フル表示・印刷

### フル表示
ファイルヘッダーの `⤢` ボタン、または `Ctrl+Shift+F` で、ヘッダー・サイドバー・各種バーを隠して本文のみを大きく表示します。再度押すか `⤡` ボタンで元に戻ります。

### 印刷 / PDF 出力
ファイルヘッダーの `⎙` ボタンでブラウザの印刷ダイアログを開きます。「PDF として保存」を選べば PDF 出力できます。印刷時はサイドバーやアウトラインなど UI 要素は除外され、本文のみが出力されます。

> 送信先は「**PDF として保存**」を選んでください。「Microsoft Print to PDF」は出力が遅く、ファイルサイズも大きくなります。

### 印刷設定
`⚙` ボタン →「印刷」タブで、印刷 / PDF 出力時の体裁を調整できます（`⎙` ボタン・`Ctrl+P` のどちらにも適用されます）。

| 設定 | 内容 |
|---|---|
| 見出しで改ページ | 選択した見出しレベル（1〜6）の 2 つ目以降の前で改ページします |
| 水平線 (`---`) で改ページする | 水平線の位置で改ページします |
| YAMLヘッダー (front matter) の後で改ページする | メタ情報を 1 ページ目に分離できます |
| コード・表・画像・図をページ途中で分割しない | 要素がページをまたがないようにします |
| 見出しをページ末尾に単独で残さない | 見出しだけがページ末尾に取り残されるのを防ぎます |
| リンクの URL を併記する | リンクテキストの後ろに URL を出力します |

> 設定を試せるサンプルとして `PRINT-SAMPLE.md` を同梱しています。

## 8. 編集・保存

### 編集モードに入る
ファイルヘッダーの「編集」ボタン（または `Ctrl+E`）でエディタが開きます。

### 分割プレビュー
編集中は左にエディタ・右にリアルタイムプレビューの分割表示になります。プレビューは入力から約 400ms 後に更新されます。

- 分割表示中はアウトライン（TOC）パネルは非表示になります。
- スクロールがエディタ・プレビュー間で連動します（比率ベース）。

### ツールバー
エディタ上部にツールバーが表示されます。選択テキストに書式を適用したり、テンプレートを挿入できます。

| ボタン | 動作 |
|---|---|
| **B** | 太字（`**text**`） |
| *I* | 斜体（`*text*`） |
| ~~S~~ | 取り消し線（`~~text~~`） |
| H1 / H2 / H3 | 見出し |
| ・ | 箇条書きリスト |
| 1. | 番号付きリスト |
| ☑ | チェックリスト |
| 🔗 | リンク挿入 |
| ` | インラインコード |
| ``` | コードブロック |
| ⊞ | テーブル雛形挿入 |

### 画像のドラッグ＆ドロップ
エディタに画像ファイルをドロップすると、ファイルと同じフォルダに保存され Markdown リンクが挿入されます。

### 保存する
「保存」ボタン（または `Ctrl+S`）で上書き保存します。

### 変更を破棄する
「破棄」ボタン（または `Esc`）で編集内容を元に戻します。確認ダイアログは表示されません。

### 未保存の確認
タブを閉じる際や全タブを閉じる際、未保存の変更があれば確認ダイアログが表示されます。

## 9. 検索

### ファイル名検索
サイドバー上部の検索欄にキーワードを入力すると、ファイル名・相対パスで絞り込みます。

- マッチしたファイルを含まないフォルダは非表示になります。
- 深い階層のファイルも検索対象に含まれます。

### 検索をクリア
検索欄を空にするか `Esc` キーで検索をクリアします。

### 全文検索
サイドバー上部の `🔍` ボタン、または `Ctrl+G` で全文検索パネルを開きます。入力欄にキーワードを入れると、ファイルの内容を対象にリアルタイムで進捗を表示しながら検索します。

- 結果一覧にはマッチした行のプレビューと強調表示が表示されます。
- ファイルをクリックすると開いて、マッチ箇所が黄色でハイライトされます。
- 再度 `🔍` ボタン（または `Ctrl+G`）を押すとパネルを閉じます。

### ページ内検索（ファイル表示中）
ファイルを開いた状態で `Ctrl+F` を押すとプレビュー内を検索するバーが表示されます。

- `Enter` / `▲` / `▼` ボタンで前後の一致箇所に移動します。
- `Esc` または `✕` で閉じます。
- マッチ件数が「3 / 12」形式で表示されます。

## 10. タグ・フラグ・メモ

ファイルごとにタグ・フラグ・メモを設定できます。設定内容はユーザーホームの `~/.mdexplorer/tags/` に保存されます（フォルダ内には作成されません）。

### タグの追加
ファイルヘッダー下のタグ入力欄にタグ名を入力して `Enter` で追加します。大文字・小文字は保持されます。

### タグの編集
タグラベルをクリックすると入力欄にタグ名が戻り、編集できます。

### タグの削除
タグの `×` ボタンで削除します。

### 重要フラグ
ファイルヘッダーの `☆` ボタン（または `Ctrl+I`）でフラグを立てます。フラグが立ったファイルは `★` で表示されます。

### メモ
タグ欄の下のメモ入力欄に自由テキストを入力できます（フォーカスを外すと自動保存）。

---

## 11. タグフィルタバー

タグやフラグでファイルを絞り込みます。

### 表示・折り畳み
タグが 1 件以上登録されると、サイドバーにタグフィルタバーが表示されます。「タグ絞込 ▾ / ▴」ヘッダーをクリックして折り畳めます。折り畳み状態は記憶されます。

### タグで絞り込む
タグチップをクリックすると選択状態（青枠）になりフィルタが適用されます。複数選択は AND 条件です。

### フラグで絞り込む
「★」ボタンで重要フラグが立ったファイルのみ表示します。

### フィルタをクリア
「✕ クリア」ボタンで全フィルタを解除します。

---

## 12. ファイル管理（作成・リネーム・削除）

ツリー上で右クリックするとコンテキストメニューが表示されます。

### 新規ファイル作成
フォルダ行またはルートフォルダ行を右クリック →「新規ファイル」。
インライン入力欄にファイル名を入力して `Enter` で作成します（`.md` は自動付与）。
現在のタグフィルタが選択中の場合、そのタグ＋「新規」タグが自動付与されます。

### 新規フォルダ作成
フォルダ行またはルートフォルダ行を右クリック →「新規フォルダ」。
フォルダ名を入力すると、同名の空 `.md` ファイルが自動生成されます（ツリーに表示するため）。

> 同名のフォルダが既に存在する場合はエラーになります。

### リネーム
ファイル行またはフォルダ行を右クリック →「リネーム」。
インライン入力欄で名前を変更して `Enter` で確定、`Esc` でキャンセルします。
拡張子（`.md`）は入力欄の外に表示されます。タグ情報はリネーム後のパスに自動移行されます。

> リネーム先と同名のファイル/フォルダが既に存在する場合はエラーになります。

### 削除
ファイル行またはフォルダ行を右クリック →「削除」。
確認ダイアログが表示されます。フォルダの場合は中身ごと削除されます。

---

## 13. 画像の表示

### 画像一覧の表示
サイドバー上部の「画像」ボタンをクリックすると、ツリーに画像ファイルもサムネイル付きで表示されます。再クリックで非表示になります。

- 対応形式: `.png` `.jpg` `.jpeg` `.gif` `.webp` `.svg` `.bmp`
- 画像のみのフォルダは「画像」表示オフ時はツリーから隠れます。

### 画像ビューワ
ツリーの画像サムネイルをクリックすると右ペインに画像ビューワが開きます。

- `＋` / `1:1` / `⊡` ボタンで拡大・等倍・フィット表示を切り替えます。
- `↗` ボタンで画像を別タブで開きます。
- `✕` ボタンでビューワを閉じます。

### Markdown 内の画像
プレビュー内の相対パス画像は自動的に解決されて表示されます。

---

## 14. HTML ファイルの表示

### HTML 一覧の表示
サイドバー上部の「HTML」ボタンをクリックすると、ツリーに `.html` / `.htm` ファイルも表示されます。再クリックで非表示になります（既定はオフ、選択は記憶されます）。

### 表示のしかた
HTML ファイルは Markdown に変換せず、そのまま **sandbox iframe** で隔離表示します。ページ自身の CSS / JavaScript がアプリの表示と衝突しません。相対パスの画像は自動的に解決されます。

### Markdown からのリンク
プレビュー内のローカル html リンク（`[説明](page.html)` など）をクリックすると、同じウィンドウ内で開きます。タブごとの戻る・進むも有効です。

> HTML ファイルは閲覧のみです。編集・タグ・被参照の表示は Markdown ファイルが対象です。

---

## 15. 被参照ファイル（バックリンク）

ファイルを開くとプレビュー下部に「被参照」バーが表示されます。現在のファイルをリンクしている他のファイル一覧が表示されます。

- `[[ファイル名]]` 形式または `[テキスト](ファイル名.md)` 形式のリンクを検索します。
- ファイル名をクリックするとそのファイルを開きます。

---

## 16. 最近開いたファイル

サイドバーに「最近開いたファイル」パネルが表示されます。

- 最大 10 件の履歴が保存されます（`localStorage`）。
- ヘッダーをクリックして折り畳めます。
- ファイル名をクリックすると開きます。

---

## 17. リンク図

ヘッダーの `🕸` ボタンで、フォルダ内 `.md` の相互リンクを図で表示します。`[リンク](file.md)` と `[[Wikiリンク]]` の参照関係を、ノード＝ファイル・矢印＝リンクとして描画します。

- **ズーム**: マウスホイール、ツールバーの `−` / `＋` / スライダー、`⤢`（全体に合わせる）
- **移動**: 図をドラッグ
- **ファイルを開く**: ノードをクリック
- **画像として保存**: 「PNG保存」ボタン
- **閉じる**: `✕` ボタンまたは `Esc`

> リンクを 1 本も持たない孤立ファイルは表示されず、件数のみツールバーに表示されます。
> ボタンを押すまで解析・描画は行われないため、通常時の負荷はありません。

## 18. 全 md を 1 つの PDF に（結合出力）

ヘッダーの `📑` ボタンで、フォルダ内の全 `.md` をツリー順に 1 つの結合プレビューにまとめます。

- **ファイル間のセパレータ**: 各ファイルの前に扉（ファイルパス・更新日時・文字数）を挿入して改ページします。
- **目次**: ファイル名を第 1 階層、各見出しを第 2 階層以降にネストした結合目次を自動生成します。
- **ファイル間リンク**: `.md` リンク / Wiki リンクは、結合ドキュメント内の該当箇所へのページ内リンクに変換されます。
- **出力**: ツールバーの `⎙ PDF出力` で印刷ダイアログを開きます（送信先は「PDF として保存」を推奨）。
- **閉じる**: `✕` ボタンまたは `Esc`

## 19. 表示設定

### フォントサイズ変更
ヘッダーの `A-` / `A+` ボタンで全体のフォントサイズを変更します。
サイズ表示（例: `16px`）をクリックするとデフォルト（16px）にリセットします。
設定は `localStorage` に保存されます。

### テーマ切替
ヘッダーの `◐` ボタンでライト / ダークテーマを切り替えます。選択したテーマは `localStorage` に保存され、再起動後も維持されます。

### サイドバー幅のリサイズ
サイドバーとコンテンツの境界をドラッグして幅を調整できます。

## 20. ステータスバーと設定パネル

### ステータスバー
画面下部のステータスバーに、起動ポート（LAN 公開時は `(LAN)` 表示）・現在開いているフォルダ・プロセス ID・サーバ稼働時間が表示されます。情報は定期的に更新されます。

### 設定パネル
ヘッダーの `⚙` ボタンで「設定とシステム情報」パネルが開きます。`×` ボタン・背景クリック・`Esc` で閉じます。「印刷」「システム情報」の 2 タブ構成です。

- **印刷**: 改ページ位置などの印刷設定（[7. フル表示・印刷](#7-フル表示印刷) を参照）
- **システム情報**
  - **サーバ**: アドレス、ネットワーク（ローカルのみ / LAN 公開）、稼働時間、現在のフォルダ
  - **プロセス**: PID、Node バージョン、プラットフォーム、配布形式（exe / Node.js）、実行ファイルパス
  - **設定ファイル**: 設定ファイル・履歴ファイルのパス

> ポートやテーマなどサーバ側の設定値は設定ファイルを直接編集し、再起動して反映します（システム情報タブは閲覧専用です）。

## 21. キーボードショートカット

| ショートカット | 動作 |
|---|---|
| `Ctrl+F` | 検索欄にフォーカス（ファイル表示中はページ内検索バーを開く） |
| `Ctrl+G` | 全文検索パネルの開閉 |
| `Ctrl+R` | ファイルツリーを再読み込み |
| `Ctrl+E` | 編集モードに入る |
| `Ctrl+S` | 保存（編集中のみ） |
| `Ctrl+I` | 重要フラグのトグル |
| `Ctrl+W` | 現在のタブを閉じる |
| `Ctrl+Tab` | 次のタブに切り替え |
| `Ctrl+Shift+Tab` | 前のタブに切り替え |
| `Ctrl+Shift+F` | フル表示のトグル |
| `Alt+←` | ナビゲーション履歴を戻る |
| `Alt+→` | ナビゲーション履歴を進む |
| `↑` / `↓` | ツリーのカーソル移動（ツリーにフォーカス時） |
| `Enter` | ツリーのカーソル位置を開く / フォルダの展開・折り畳み |
| `→` / `←` | ツリーのフォルダを展開 / 折り畳み |
| `Esc` | 編集を破棄 / 検索をクリア / ページ内検索・設定パネル・メニュー・オーバーレイを閉じる |
