## 1. フォルダを開く

### パスを直接入力する
ヘッダーのパス入力欄にフォルダのパスを入力して `Enter` または「開く」ボタンを押します。

### フォルダ参照ダイアログを使う
`📂` ボタンをクリックするとOS標準のフォルダ選択ダイアログが開きます。

### 履歴から選ぶ
「履歴 ▾」ドロップダウンに過去に開いたフォルダが記録されています。選択するとそのまま開きます。

### ドラッグ＆ドロップ
フォルダをブラウザウィンドウにドロップしても開けます。

> Markdown ファイルが 1 件もないフォルダはツリーに表示されません。

---

## 2. ファイルツリー

左のサイドバーにフォルダ構造とファイル一覧が表示されます。

### ディレクトリの展開・折り畳み
フォルダ行をクリックすると展開・折り畳みを切り替えられます。

### ルートフォルダ行
ツリー最上部のフォルダ名（太字・アクセントカラー）はルートフォルダを示します。右クリックでルート直下に新規作成できます。

### フォルダバッジ
フォルダアイコンの横に `md` / `img` のバッジが表示されます。それぞれ Markdown ファイル・画像ファイルを含んでいることを示します（サブフォルダを含む再帰的な判定）。

### ファイルカウント
サイドバー下部に総ファイル数が表示されます。フィルタ中は「表示件数 / 総数」になります。

### ツリーの再読み込み
ヘッダーの `↺` ボタン（または `Ctrl+R`）でツリーを再スキャンします。外部でファイルを追加・削除した場合に使います。

---

## 3. ファイルを開く・タブ操作

### ファイルを開く
ファイル行をクリックするとタブで開きます。

### タブの切り替え
タブバーのタブをクリックして切り替えます。`Ctrl+Tab` / `Ctrl+Shift+Tab` でも切り替えられます。

### タブを閉じる
タブの `×` ボタン、または `Ctrl+W` で閉じます。

### 全タブを閉じる
タブバー右端の「全て閉じる」ボタンで一括クローズします。未保存の変更があれば確認ダイアログが表示されます。

### タブ状態の保存
フォルダを切り替えても、戻った時に以前のタブが復元されます（`localStorage` に保存）。

---

## 4. プレビュー表示

ファイルを開くと Markdown がレンダリングされて表示されます。

### Mermaid ダイアグラム
コードブロックに` ```mermaid ` で記述したダイアグラムが自動でレンダリングされます。

- **ズーム**: ダイアグラム上でマウスホイールを回す
- **パン（移動）**: ダイアグラムをドラッグ
- **ズームリセット**: ダイアグラム右上の `↺` ボタン
- **拡大・縮小ボタン**: ダイアグラム右上の `+` / `−`

### コードハイライト
コードブロックはシンタックスハイライト表示されます（テーマに連動）。

### コードブロックのコピー
コードブロックにマウスを合わせると右上に「コピー」ボタンが表示されます。クリックでクリップボードにコピーされ、「✓」表示に変わります。

### テーブルのソート
テーブルのヘッダー行をクリックするとその列で昇順・降順ソートできます。`↑` / `↓` が列名の後ろに表示されます。

## 5. 編集・保存

### 編集モードに入る
ファイルヘッダーの「編集」ボタン（または `Ctrl+E`）でテキストエリアが開きます。

### 分割プレビュー
「分割」ボタンで左にエディタ・右にリアルタイムプレビューの分割表示になります。プレビューは入力から約 400ms 後に更新されます。

- 分割表示中はアウトライン（TOC）パネルは非表示になります。
- スクロールがエディタ・プレビュー間で連動します（比率ベース）。

### ツールバー
エディタ上部にツールバーが表示されます。選択テキストに書式を適用したり、テンプレートを挿入できます。

| ボタン | 動作 |
|---|---|
| **B** | 太字（`**text**`） |
| *I* | 斜体（`*text*`） |
| ~~S~~ | 取り消し線（`~~text~~`） |
| H1 / H2 / H3 | 見出し |
| UL | 箇条書きリスト |
| OL | 番号付きリスト |
| ☑ | チェックリスト |
| Link | リンク挿入 |
| Code | インラインコード |
| ``` | コードブロック |
| Table | テーブル雛形挿入 |

### 画像のドラッグ＆ドロップ
エディタに画像ファイルをドロップすると、ファイルと同じフォルダに保存され Markdown リンクが挿入されます。

### 保存する
「保存」ボタン（または `Ctrl+S`）で上書き保存します。

### 変更を破棄する
「破棄」ボタン（または `Esc`）で編集内容を元に戻します。確認ダイアログは表示されません。

### 未保存の確認
タブを閉じる際や全タブを閉じる際、未保存の変更があれば確認ダイアログが表示されます。

## 6. 検索

### ファイル名検索
サイドバー上部の検索欄にキーワードを入力すると、ファイル名・相対パスで絞り込みます。

- マッチしたファイルを含まないフォルダは非表示になります。
- 深い階層のファイルも検索対象に含まれます。

### 検索をクリア
検索欄を空にするか `Esc` キーで検索をクリアします。

### 全文検索
サイドバー上部の検索欄で `Ctrl+Enter` を押すと全文検索に切り替わります。ファイルの内容を対象にリアルタイムで進捗を表示しながら検索します。

- 結果一覧にはマッチした行のプレビューと強調表示が表示されます。
- ファイルをクリックすると開いて、マッチ箇所が黄色でハイライトされます。

### ページ内検索（ファイル表示中）
ファイルを開いた状態で `Ctrl+F` を押すとプレビュー内を検索するバーが表示されます。

- `Enter` / `▲` / `▼` ボタンで前後の一致箇所に移動します。
- `Esc` または `✕` で閉じます。
- マッチ件数が「3 / 12」形式で表示されます。

## 7. タグ・フラグ・メモ

ファイルごとにタグ・フラグ・メモを設定できます。設定内容はユーザーホームの `~/.mdexplorer/tags/` に保存されます（フォルダ内には作成されません）。

### タグの追加
ファイルヘッダー下のタグ入力欄にタグ名を入力して `Enter` で追加します。大文字・小文字は保持されます。

### タグの編集
タグラベルをクリックすると入力欄にタグ名が戻り、編集できます。

### タグの削除
タグの `×` ボタンで削除します。

### 重要フラグ
ファイルヘッダーの `☆` ボタン（または `Ctrl+I`）でフラグを立てます。フラグが立ったファイルは `★` で表示されます。

### メモ
タグ欄の下のメモ入力欄に自由テキストを入力できます（フォーカスを外すと自動保存）。

---

## 8. タグフィルタバー

タグやフラグでファイルを絞り込みます。

### 表示・折り畳み
タグが 1 件以上登録されると、サイドバーにタグフィルタバーが表示されます。「タグ絞込 ▾ / ▴」ヘッダーをクリックして折り畳めます。折り畳み状態は記憶されます。

### タグで絞り込む
タグチップをクリックすると選択状態（青枠）になりフィルタが適用されます。複数選択は AND 条件です。

### フラグで絞り込む
「★」ボタンで重要フラグが立ったファイルのみ表示します。

### フィルタをクリア
「✕ クリア」ボタンで全フィルタを解除します。

---

## 9. ファイル管理（作成・リネーム・削除）

ツリー上で右クリックするとコンテキストメニューが表示されます。

### 新規ファイル作成
フォルダ行またはルートフォルダ行を右クリック →「新規ファイル」。
インライン入力欄にファイル名を入力して `Enter` で作成します（`.md` は自動付与）。
現在のタグフィルタが選択中の場合、そのタグ＋「新規」タグが自動付与されます。

### 新規フォルダ作成
フォルダ行またはルートフォルダ行を右クリック →「新規フォルダ」。
フォルダ名を入力すると、同名の空 `.md` ファイルが自動生成されます（ツリーに表示するため）。

> 同名のフォルダが既に存在する場合はエラーになります。

### リネーム
ファイル行またはフォルダ行を右クリック →「リネーム」。
インライン入力欄で名前を変更して `Enter` で確定、`Esc` でキャンセルします。
拡張子（`.md`）は入力欄の外に表示されます。タグ情報はリネーム後のパスに自動移行されます。

> リネーム先と同名のファイル/フォルダが既に存在する場合はエラーになります。

### 削除
ファイル行またはフォルダ行を右クリック →「削除」。
確認ダイアログが表示されます。フォルダの場合は中身ごと削除されます。

## 10. 画像の表示

### 画像一覧の表示
サイドバー上部の「画像」ボタンをクリックすると、ツリーに画像ファイルもサムネイル付きで表示されます。再クリックで非表示になります。

- 対応形式: `.png` `.jpg` `.jpeg` `.gif` `.webp` `.svg` `.bmp`
- 画像のみのフォルダは「画像」表示オフ時はツリーから隠れます。

### 画像ビューワ
ツリーの画像サムネイルをクリックすると右ペインに画像ビューワが開きます。

- `+` / `−` ボタンで拡大・縮小します。
- 赤い `✕` ボタンでビューワを閉じます。

### Markdown 内の画像
プレビュー内の相対パス画像は自動的に解決されて表示されます。

---

## 11. 被参照ファイル（バックリンク）

ファイルを開くとプレビュー下部に「被参照」バーが表示されます。現在のファイルをリンクしている他のファイル一覧が表示されます。

- `[[ファイル名]]` 形式または `[テキスト](ファイル名.md)` 形式のリンクを検索します。
- ファイル名をクリックするとそのファイルを開きます。

---

## 12. 最近開いたファイル

サイドバーに「最近開いたファイル」パネルが表示されます。

- 最大 10 件の履歴が保存されます（`localStorage`）。
- ヘッダーをクリックして折り畳めます。
- ファイル名をクリックすると開きます。

---

## 13. 表示設定

### フォントサイズ変更
ヘッダーの `A-` / `A+` ボタンで全体のフォントサイズを変更します。
サイズ表示（例: `16px`）をクリックするとデフォルト（16px）にリセットします。
設定は `localStorage` に保存されます。

### テーマ切替
ヘッダーの `◐` ボタンでライト / ダークテーマを切り替えます。

### サイドバー幅のリサイズ
サイドバーとコンテンツの境界をドラッグして幅を調整できます。

---

## 14. キーボードショートカット

| ショートカット | 動作 |
|---|---|
| `Ctrl+F` | 検索欄にフォーカス（ファイル表示中はページ内検索バーを開く） |
| `Ctrl+Enter` | 全文検索を実行（検索欄にフォーカス中） |
| `Ctrl+R` | ファイルツリーを再読み込み |
| `Ctrl+E` | 編集モードに入る |
| `Ctrl+S` | 保存（編集中のみ） |
| `Ctrl+I` | 重要フラグのトグル |
| `Ctrl+W` | 現在のタブを閉じる |
| `Ctrl+Tab` | 次のタブに切り替え |
| `Ctrl+Shift+Tab` | 前のタブに切り替え |
| `Esc` | 編集を破棄 / 検索をクリア / メニューを閉じる |
