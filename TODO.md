# fastmd-explorer 機能追加チェックリスト

## 実装順（難易度順）

- [x] コードブロック コピーボタン — レンダリング後に DOM 操作するだけ、サーバー変更なし
- [x] ページ内検索（Ctrl+F） — プレビュー内テキストを走査、全文検索の仕組みを流用
- [x] テーブルソート — レンダリング済みテーブルへの DOM 操作
- [x] 最近開いたファイル — クライアント状態 + localStorage のみ
- [x] スクロール同期 — 編集モードでエディタ↔プレビューのスクロール連動（分割表示）
- [x] Markdown ツールバー — 編集モードに太字・見出し・リンク等のボタン
- [x] バックリンク表示 — 全文検索インフラを流用して参照元ファイルを一覧表示
- [x] 画像ドラッグ&ドロップ挿入 — エディタへの画像ドロップで `![]()` 自動挿入
